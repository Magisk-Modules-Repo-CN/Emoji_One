# EmojiOne

在System-less下替换Emoji为EmojiOne。

*当前EmojiOne版本: `4.0` ([EmojiOne源代码](https://github.com/emojione/emojione-assets/releases/download/4.0/emojione-android.ttf))*

## 已知的不兼容性
* 它将会修改系统的Emoji字体文件。
* **可能无法正常应用于某些OEM厂商的Android设备，例如HTC设备!**
